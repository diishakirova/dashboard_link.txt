# TED Dashboard Analysis

## О проекте
Разработка аналитического дашборда для компании, организующей конференции TED.

Цель проекта — использовать данные прошлых мероприятий для:
- выбора тем выступлений  
- подбора спикеров  
- планирования длительности конференции  
- анализа интереса аудитории  

---

##  Данные
Использованы таблицы:

- **events** — информация о конференциях  
- **talks** — данные о выступлениях  
- **speakers** — информация о спикерах  

---

##  Задачи анализа
Дашборд отвечает на ключевые бизнес-вопросы:

- Какие темы популярны и как на них реагируют зрители  
- Среднее количество выступающих на конференции  
- Средняя длительность выступлений  
- Самые популярные и самые «смешные» выступления  
- Топ тегов выступлений  
- Какие спикеры получают больше всего аплодисментов  

---
